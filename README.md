# Sistema de Gestión de Clínica

## 1. Portal del Paciente

### Autenticación
- Registro mediante correo y contraseña.
- Inicio de sesión.

### Reservar cita
- Seleccionar especialidad.
- Seleccionar médico.
- Mostrar calendario con horarios disponibles (bloques de 30 minutos).
- Escribir motivo de la consulta.
- Confirmar la cita.

### Mis citas
- Ver próximas citas.
- Ver historial de citas.

### Cancelación de citas
- Permitir cancelar únicamente si falta más de un día para la consulta.
- Si falta menos de un día, mostrar un mensaje indicando que debe comunicarse con la clínica.

---

## 2. Portal del Médico

### Inicio de sesión
- Acceso mediante cuentas creadas por el administrador.

### Agenda
- Calendario semanal.
- Horario de atención:
  - Lunes a Viernes
  - 8:00 AM - 6:00 PM
- Mostrar:
  - Paciente
  - Hora
  - Motivo de consulta
- Permitir:
  - Marcar consulta como atendida.
  - Agregar notas médicas.

### Historial de consultas
- Buscar por nombre del paciente.
- Consultar consultas anteriores.
- Visualizar notas médicas.

---

## 3. Portal del Administrador

### Dashboard
- Mostrar número de citas agendadas para el día.

### Gestión de médicos
- Alta de médicos.
- Baja de médicos.
- Asignación de especialidades.

### Gestión de citas
- Visualizar agenda completa.
- Reagendar citas.
- Cancelar citas manualmente.
